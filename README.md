# Aprende Brincando! 🎨 (Suíte Educativa Interativa)

Um portal educativo e interativo projetado para crianças e estudantes aprenderem **Horas**, **Dias da Semana**, **Meses** e **Estações do Ano** em **Português Europeu (`pt-PT`)**.

![Aprende Brincando](https://img.shields.io/badge/Language-Portuguese%20%28pt--PT%29-blue)
![License](https://img.shields.io/badge/License-MIT-green)

---

## 🎨 Módulos Educativos

1. ⏰ **Aprende as Horas!** (`horas.html`)
   - Relógio analógico e digital Canvas interativo com ponteiros arrastáveis, voz `pt-PT` e galeria de conquistas.
2. 📅 **Aprende os Dias da Semana** (`dias.html`)
   - Ordenação dos 7 dias da semana, distinção entre dias úteis e fim de semana, pronúncia e quizes.
3. 🗓️ **Aprende os Meses do Ano** (`meses.html`)
   - Os 12 meses do ano, calendário de 30/31 dias, feriados e épocas festivas.
4. 🍂 **Aprende as Estações do Ano** (`estacoes.html`)
   - As 4 estações (Primavera 🌸, Verão ☀️, Outono 🍁, Inverno ❄️), clima, meses e quizes visuais.

---

## 🚀 Como Executar Localmente

Execute o servidor HTTP local:
```powershell
python -m http.server 8000 --directory web
```
E aceda a **`http://localhost:8000`** no seu navegador!

---

## 🐍 Aplicação Desktop Kivy (Python)
Localizada na raiz (`time_teacher_app.py`):
```powershell
python time_teacher_app.py
```
